## 🔧 Local Playback

인터넷이 불안정한 발표장을 대비해 **로컬 재생**도 지원합니다.
이 레포지토리를 다운로드(또는 clone)한 뒤 `index.html` 파일을 더블클릭하면 동일한 인터페이스로 동작합니다.

```bash
git clone https://github.com/kkhgeo/digital-eia-kr.git
cd digital-eia-kr
# index.html 더블클릭
```
